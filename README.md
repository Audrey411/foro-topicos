# ForoHub - Topicos  

### Descripción del Proyecto:

ForoHub - Topicos es una aplicación de consola desarrollada con Spring Boot, que permite ejecutar un foro de discusion con funcionalidades CRUD,  autenticación y autorización usando JWT y Spring Security, todo esto mediante la creacion de API rest.

### Requisitos previos:
- Java: JDK 17 o superior
- Spring Boot: versión 3.3.5
- SQL Server: como base de datos
- Spring Tools IDE: para desarrollo y ejecución
- Configuración en application.properties: archivo de configuración para conexión a la base de datos, secret key de JWT y otros parámetros de autenticación.

### Tecnologías utilizadas:
- Java 17
- Spring Boot 3.4.1
- Maven
- MySQL Workbench
- Jackson
- Http
- Insomnia
- JPA / Hibernate

### Funcionalidades:
- Autenticacion para usuarios.
- Registrar Topico.
- Actualizar Topico.
- Listar Topicos.
- Eliminar Topico.
